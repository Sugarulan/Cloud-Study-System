# 6 周路线图（方雨菲视角）

> 总体项目周期压缩到 **6 周**，方雨菲负责 **后期集成与测试阶段**。
> 6 周内的整体策略：**W1 基座 + W2 后端 + W3 学员端 + W4 AI + W5 测试 + W6 交付**。

---

## W1｜基座与契约（第 1 周）

**目标**：项目骨架跑起来 + AI 适配层就位 + 接口契约冻结

### 你的任务
- 后端多模块脚手架（5 个业务模块空壳）
- Spring AI 适配层（Provider 接口 + Mock）
- 前端 Vue3 脚手架（学员端 3 个空页面）
- Docker Compose（MySQL / Redis / MinIO）
- 接口契约文档

### 交付物
- ✅ 后端 `mvn spring-boot:run` 可启动
- ✅ 前端 `npm run dev` 可启动
- ✅ AI 调用 Mock 通
- ✅ `docs/api-contract.md` 与王茗瑾评审通过

---

## W2｜后端五大模块（第 2 周）

**目标**：评卷 / 成绩 / 在线作答 / 个人测评 / 知识管理 全部接口 + 业务 + 单测

### 你的任务
| 模块 | WBS | 关键能力 |
|------|-----|----------|
| 评卷 | 3.3.6 | 客观题自动判分、主观题 AI 评分、人工评卷、复核、发布 |
| 成绩 | 3.3.7 | 多条件筛选、统计、Excel 导出 |
| 在线作答 | 3.3.8 | 试卷渲染、Redis 暂存、断点续答、计时、自动交卷 |
| 个人测评 | 3.3.9 | 待考 / 已考列表、错题本、AI 解析调用 |
| 知识管理 | 3.3.10 | 目录 / 标签、RBAC、状态机、版本控制 |
| 文件服务 | 3.3.4 | MinIO 封装、统一消息中心 |

### 依赖
- 王茗瑾 W2 末完成试卷 / 考试模块，否则你的"在线作答"无法联调

### 交付物
- Swagger 上 5 个模块接口齐全
- Service 单测覆盖率 ≥60%
- 接口契约锁定（冻结后不再改字段）

---

## W3｜学员端 UI 攻坚（第 3 周）

**目标**：3 个学员端核心页面 + 主链路联调

### 你的任务
- **在线作答页面**：计时器、切题、暂存提示、自动交卷倒计时、客观题 + 主观题作答
- **个人测评页面**：待考列表、已考列表、错题本、AI 解析展示
- **知识管理页面**：富文本编辑器、版本对比、目录树
- 前后端联调（与王茗瑾的管理端打通）
- 安全测试（SQL 注入 / XSS / 越权）

### 交付物
- 学员能完成"登录 → 进入考试 → 作答 → 交卷 → 查看成绩 → 查看错题"主链路
- 安全测试报告

---

## W4｜AI 与集成（第 4 周）

**目标**：AI 评卷 + AI 错题解析 + AI 文档抽题 全部跑通

### 你的任务
- Spring AI 真实 Provider 接入（企业内网 REST API）
- AI 评卷 Prompt 设计与调试
- AI 错题解析
- AI 文档 → 题目抽取（PDF / Word / Markdown）
- Resilience4j 熔断 + 限流
- AI 服务降级（切回人工评卷）
- Prompt 版本管理

### 关键原则
**AI 故障不影响主流程** —— Mock Provider + 降级策略必须先于真实接入上线。

### 交付物
- AI 评卷 + 错题解析可在演示场景跑通
- AI 故障时降级可用（人工评卷）
- Prompt 版本库

---

## W5｜测试与质量（第 5 周）

**目标**：测试覆盖率 ≥80% + 性能压测 500 并发 + Bug 收敛

### 你的任务
- 单测覆盖率从 60% 提升到 ≥80%
- Testcontainers 集成测试
- 性能压测（重点是**在线作答**，500 并发）
- 兼容性测试（Chrome / Edge / Firefox）
- 回归测试 + Bug 修复
- 缺陷管理（Bug 生命周期跟踪）
- 前端性能优化（首屏 < 2s）

### 交付物
- 性能压测报告
- Bug 清单收敛到 P0 已修复
- 前端 Lighthouse 报告

---

## W6｜交付与培训（第 6 周）

**目标**：交付物齐全 + 培训完成 + UAT 通过

### 你的任务
- 学员操作手册（PDF + 在线）
- 运维手册（与王茗瑾协作）
- API 文档完善（Swagger）
- 培训视频录制（学员端）
- UAT 用户验收测试
- 联合部署 / 灰度发布

### 交付物
- 学员手册 / 培训视频
- 最终部署包
- UAT 验收报告
- 项目结项报告（你的部分）

---

## 风险与应对

| 风险 | 应对 |
|------|------|
| 王茗瑾进度延迟 | W1 末冻结接口契约，你按契约并行开发 |
| AI 模型效果差 | Mock Provider 兜底；熔断降级先于 AI 上线 |
| 在线作答并发问题 | Redis Hash 存快照 + 版本号防覆盖 |
| 6 周塞 18 周内容 | P0 优先，P1 降级（如版本对比、Prompt 评估可后置） |
| 学员端 UI 工作量低估 | Element Plus 现成组件，不造轮子 |
