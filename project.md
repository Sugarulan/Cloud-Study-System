广汽云学习管理系统（GAC-LMS）。

# 模块

+ 账号管理模块：支持账号创建、更新、停用的操作，角色管理（权限配置、角色分配）、密码管理（密码策略、密码重置）
+ 人员信息管理模块：支持人员信息创建、更新、修改、和账号关联等操作，支持根据系统、部门等条件将人员分组，支持人员导入导出。
+ 题目管理模块：题目创建、编辑、删除的管理界面以及题目属性（难度、分类、标签）、题目答案管理页面、支持批量操作，批量导入导出。
+ 试卷管理模块：试卷创建、编辑、删除、发布的功能，可以通过抽取题目组建试卷的功能。
+ 考试管理模块：考试创建、考试编辑、考试发布、交卷规则设置（设置考试的自动交卷/手动交卷规则）、考试规则设置（参考人员范围等）、考试周期管理（设置考试的开始/结束时间）的功能。
+ 评卷模块：支持自动评阅、人工评阅、分数计算、评卷复核、成绩发布的功能
+ 成绩管理模块：成绩查看、统计（统计数据分析，分数分布/通过率/平均分）、筛选（多条件筛选）的管理页面，支持成绩报表和成绩导出Excel表格。
+ 个人测评管理模块：在线考试（参加在线考试的入口）、考试记录（查看未参加和已参加的考试列表）、错题查看和解析的功能。
+ 在线作答模块：支持试卷渲染、用户答题操作（选择、输入）、答题中途暂存、答题计时、断点续答、交卷确认的功能
+ 系统集成模块：为系统配置AI大模型接入，与Al大模型集成，通过AI的能力为考题结果进行自动评价，为错题的详细解析的能力；支持与邮件、webhook等集成，实现考试提醒功能。
+ 知识管理模块：知识库功能，支持知识库的目录、分类和标签内容、知识库角色和权限等内容；支持收集、整理、记录、编排各个系统的操作使用手册、功能文档、设计文档、运维文档等；建立“创建-审核-发布-更新-归档”的管理闭环流程；支持通过接入的 AI大模型将文档内容抽取为题目。
